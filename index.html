สูตรที่ใช้คำนวณ (ต่อ order)

🔹 ค่าสินค้า = Σ ( ราคา/ชิ้น × จำนวนใน order นี้ )

🔹 ค่าส่งจีน = Σ ( ค่าส่งจีนรวม[สินค้า] ÷ จำนวนรวมทุก order[สินค้า] × จำนวนใน order นี้ )

🔹 ค่า Import = Σ ( ค่า import รวม[สินค้า] ÷ จำนวนรวมทุก order[สินค้า] × จำนวนใน order นี้ )

🔹 ค่าส่งไทย = คงที่ (เท่ากันทุก order)

💰 รวม = ค่าสินค้า + ค่าส่งจีน + ค่า Import + ค่าส่งไทย

⚠️ order ที่รวมได้ 0 จะไม่ถูกบันทึก
